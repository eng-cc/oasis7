## 视觉方向
- 采用 `offer decision memo / 决策档案` 方向：像一份被摊开的选择备忘录，而不是校园招聘海报或 AI 科技海报。
- 每页都应该像同一份决策文件里的不同章节：封面、判断原则、价值拆解、最后的选择题。
- 重点不是“哪个岗位更酷”，而是“第一份工作先拿什么结构性的成长”。
- 可用的视觉暗示：
  - 决策标签、分栏 memo、编号清单、评估印章、A/B 选择卡
  - 奶油纸面、深墨蓝、砖红色提示、偏文件夹式结构线
  - 强标题 + 短判断句，不做技术炫光
- 不要出现：
  - AI 芯片、脑机接口、蓝紫赛博光效
  - 校招宣讲会海报、招聘启事版式
  - 夸张的薪资、offer 数量、岗位表格

- 综合评分：`8.9/10`
