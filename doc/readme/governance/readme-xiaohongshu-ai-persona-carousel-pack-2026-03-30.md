## 逐页文案

### 第 1 页 / 封面
```text
AI人格很火

但我不想做
陪聊搭子

我更想做
会在世界里自己行动的角色
```

### 第 2 页 / 核心判断
```text
人格不只是
会聊天

如果它只会在聊天框里
把话说得像真人

那还是太薄了

我更在意的是
它进了世界以后
会怎么选
会怎么做
会留下什么后果
```

### 第 3 页 / 具体例子
```text
比如这些时候
它会怎么选

资源紧张
先保能源还是先保数据

交易吃亏
下次会不会更谨慎

同伴掉链子
会补位 甩锅
还是干脆退出
```

### 第 4 页 / 收束页
```text
所以我更想要的

不是更会聊天的搭子

而是进了世界以后
真的会活出自己倾向的角色

如果是你
你会更想要哪一种？
```

## 评论区引导
