## 逐页文案

### 第 1 页 / 封面
```text
今年春招，
我会看什么人

不先看你会多少工具
先看你能不能把事接住
```

### 第 2 页 / 判断前提
```text
AI 把“会一点”的人
变多了

现在真正稀缺的
反而不是“都会一点”

而是
能把事接住的人
```

### 第 3 页 / 前两条标准
```text
我先看两件事

讲得清
做得完

不是“参与过”
是你具体做了什么

不是项目有多大
是你有没有认真收过尾
```

### 第 4 页 / AI 使用方式
```text
“我会用 AI”
已经不算亮点了

我更在意的是

你拿它代打

还是拿它提速
但最后自己判断
自己验收
```

### 第 5 页 / 玩家感觉与问题处理
```text
还有两条

有没有玩家感觉
出问题会不会往前推

做游戏不是堆功能
是你知不知道玩家会不会点

做项目也一样
不是只会报问题
是能不能先给一个能落地的方案
```

### 第 6 页 / 收束页
```text
所以我最后会看

判断
执行
收口

如果你现在还在春招
你觉得自己最强的是哪一项？
```

## 评论区引导
